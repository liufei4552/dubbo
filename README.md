# dubbo简单例子 
##环境：
   * _jdk1.8+dubbo2.5.3+spring4.2.5.RELEASE+zookeeper3.4.13+Tomcat8.5_
#####1.首先加入了服务提供者也就是生产者:
   * 主要运用maven创建聚合项目
   * 搭建环境时一定要注意jar的引用，本人在此耽误很多时间
   
#####2.在自己环境配置了dubbo-admin:
   * 服务管理页面可以查看自己提供的服务及状态(dubbo-admin-2.5.10.war)
   * dubbo2.6以上就没有该功能了
   * 注意更改配置文件中zk的地址
#####3.加入消费者
   * 消费者需要引入服务可以使用注解和手动引入，工作中一般使用手动引入
   * 最后在单元测试中可以看到
